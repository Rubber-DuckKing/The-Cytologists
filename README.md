# Cellularis

A wave-survival auto-battler built in Godot where you guide a main cell, split into clones, and stack upgrades and mutations to outscale incoming enemies.

## Overview

- Survive escalating enemy waves.
- Your cells attack automatically when enemies enter range.
- Level up to choose upgrades for individual cells.
- Build around splitting, survivability, bombardment, poison, and mutations.

## Controls

- `Mouse`: Move the main cell
- `Tab`: Toggle inspect mode
- `Upgrade Buttons`: Select which cell receives the next upgrade

## Debug Shortcuts

- `F6`: Advance to the next wave
- `F7`: Open the upgrade screen

## Current Features

- Cell splitting through `Mitosis` and `Fracture`
- Per-cell upgrades such as `Beefy`, `Bombardment`, `Gang Up`, `Devour`, `Mitochondria`, and `Accelerant`
- Random mutation system with eukaryote progression
- Virus enemy with poison projectiles
- Inspect UI for checking a cell's current traits

## Tech

- Engine: Godot `4.6.1`
- Main project name: `cellularis`
